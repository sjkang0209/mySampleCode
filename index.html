DEVELOPER

My name is Sujeong Kang and I develop websites through coding using web technologies such as HTML, CSS, Sass, jQuery and some of PHP. I started in the web design industry three years ago from South Korea and I am currently studying interactive design from Media Design School in New Zealand.
I like to resolve design problems, create smart user interfaces and imagine useful interactions. I have been creating user-friendly design ranging from web to mobile.

WORK

  • su-jeong.com/mamas

    HTML Sass jQuery Handlebar PHP

    From the minor project presentation in graphic design class, I was inspired by Eliza’s work so I took her work for my major project, which is about food delivery within central Auckland. Most of the concepts like color, fonts and brand identity are from Eliza’s minor project. I used PHP for account login, sign-up and cart page. Users can order their food for pick up or request a delivery through the website. The pages with forms are all in “template.js”.

  • su-jeong.com/huiahops

    HTML Sass jQuery

    This website is created with Megan from graphic design class who came up with beer brand design. I used logo, mock up images and brand history from her works in my website. Because we had to build new brand from the scratch, we couldn’t make a lot of contents. So I made the site to function without scroll and gave it sliding effect to display products as a gallery. It works for all sizes and shows animation when it’s over the tablet size.

  • su-jeong.com/lingerie

    HTML Sass jQuery WordPress Pod

    In this project I wanted to challenge myself by creating an online shopping mall site by using ‘Word press’. As this website sells lingerie for women, I mainly used feminine colors. I created product list pages so it can be filtered, detail pages for each of products to have diverse options and cart page was added to remove or save items. Also shop owner can pick and display the best items amongst their products in the landing page. This website works fine in most of devices.

  • su-jeong.com/drama

    HTML CSS jQuery PHP

    This website is created for the fans of American drama, “Madam Secretary”. To make accordance with the drama’s atmosphere, I chose the dark blue color with yellow to give highlights throughout the site. I displayed several trailer videos for each of episodes with context and introduced main casts’ information. Throughout this project I mainly used PHP to create data and media query for a responsive site based on a mobile first coding.

  • su-jeong.com/designer

    HTML CSS jQuery

    I made this website to introduce the designers from IKEA in a single page. I matched color theme to IKEA logo and tried to give brand concept through the short video with a slogan in the landing page. The background image in each of designer’s biography section is the most famous product from their career. Also I used media query to make the site responsive and fit different screen sizes.

  • su-jeong.com/biogaia

    HTML CSS jQuery

    While I was working, my last job was to make a new website for the brand new medicine, which was being imported from Sweden and sold by the company in Korea. Because my supervisor gave me two weeks for creating the site, I put all the contents in a single page, do all of design and coding. I followed the color scheme from brand identity made by head office and made the website work in most of screen sizes.
